## 2. 使用 art-template 抽取布局部分

### 2.1. 说明

文档：https://aui.github.io/art-template/zh-cn/docs/syntax.html

#### 2.1.1. 子模板

**标准语法**

用于引入其他模板。

```html
{{ include './header.html' }}
或
{{ include './header.html' data }}
```

#### 2.1.2. 模板继承

layout.html：

```html
<html lang="en">
<head></head>
<body>
  {{ include './header.html' }}

  <!-- 留坑 -->
  {{ block 'content' }} 默认 content 内容 {{ /block }}

  {{ include './footer.html' }}
</body>
</html>
```

index.html：

```html
<!-- 继承 layout.html -->
{{ extend './layout.html' }}

<!-- 填坑 -->
{{ block 'content' }} index 内容 {{ /block }}
```
### 2.2. 目录

```
blog/
  views/    # 视图目录
    layout/
      header.html
      footer.html
      default.html
    index.html
```

### 2.3. 默认布局

blog/layout/header.html :

```html
<div>
  <h1>header</h1>
</div>
```

blog/layout/footer.html：

```html
<div>
  <h1>footer</h1>
</div>
```

blog/layout/default.html：

```html
<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <title>{{block 'title'}}My Site{{/block}}</title>

  <link rel="stylesheet" href="/public/lib/bootstrap/css/bootstrap.css">
  <link rel="stylesheet" href="/public/css/main.css">

  <!-- 留坑 -->
  {{ block 'head' }} 默认 head 内容 {{ /block }}

</head>
<body>

  {{ include './header.html' }}

  <!-- 留坑 -->
  {{ block 'content' }} 默认 content 内容 {{ /block }}

  {{ include './footer.html' }}

  <script src="/public/lib/jquery.js"></script>
  <script src="/public/lib/bootstrap/js/bootstrap.js"></script>

  <!-- 留坑 -->
  {{ block 'script' }} 默认 script 内容 {{ /block }}

</body>
</html>
```

### 2.4. 使用默认布局

blog/index.html：

```html
{{ extend './layout/default.html'}}

{{ block 'title' }}{{ title }}{{ /block }}

{{ block 'head' }}
<style>
  body {
    color: #333;
  }
</style>
{{ /block }}

{{ block 'content' }}
index 内容
{{ /block }}

{{ block 'script' }}
<script>
  console.log('script');
</script>
{{ /block }}
```
