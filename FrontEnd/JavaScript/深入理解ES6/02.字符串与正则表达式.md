# 第二章 字符串与正则表达式

ES6加强了字符串和正则表达式。

## 1. 识别字符串的方法

* includes()
* startsWith()
* endsWith()

## 2. repeat() 方法

```javascript
"x".repeat( 3 ); //=> "xxx"
```

可应用于需要产生缩进的地方。

## 3. 模板字面量

### 3.1. 语法

使用反引号包裹普通字符串：

```javascript
`hello world!`
```

使用反斜杠（`\`）转义字符串中的反引号：

```javascript
`hello \`world!`
```

### 3.2. 多行字符串

```javascript
let html = `
    <ul>
        <li><a href="#">首页</a></li>
        <li><a href="#">关于</a></li>
    </ul>
`
```

### 3.3. 替换位

替换位允许将有效的 JS表达式 嵌入到模板字面量。

界定符：`${` 和 `}`。

```javascript
let name = "吴钦飞";
`Hello, ${name}`
```
