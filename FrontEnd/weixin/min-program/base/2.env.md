# 上手第一个小程序

## 1. 开发前的准备

1. [注册小程序账号](https://mp.weixin.qq.com/)
2. 激活邮箱
3. 信息登记
4. 登陆小程序管理后台
5. 完善小程序信息
6. 绑定开发者

## 2. 小程序代码结构和基本配置

```text
${root}/
  app.js      # 注册 app
  app.json    # app 全局配置
  app.wxss    # app 全局样式
  project.config.json   # 开发者工具的配置文件
  pages/      # 所有小程序界面
    index/      # 主页界面
      index.js    # 逻辑
      index.wxml  # 内容
      index.wxss  # 样式
  utils/

```
