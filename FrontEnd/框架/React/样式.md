# 样式

## 1. 行内样式

```jsx
class CommentList extends React.Component {
  constructor() {
    super();

    this.state = {
      msg: '评论列表',
    };
  }

  render() {
    const { msg } = this.state;

    return <div>
      <h1 style={ { color: 'red', fontSize: '18px' } }>{ msg }</h1>
    </div>;
  }
}
```

行内样式不能使用字符串值，需要使用样式对象。
