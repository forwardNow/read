## 2. 安装所需的插件

### 2.1. 安装 react-router、axios

```shell
# 安装与 HTML 混合的 router 即可
$ yarn add react-router-dom

# HTTP 插件

└─ axios@0.18.0
```

### 2.4. 安装与配置 less-loader

```shell
yarn add --dev less-loader
```
