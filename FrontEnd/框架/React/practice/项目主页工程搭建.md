### 2.4. 安装 AntD

>官网：[https://ant.design/index-cn](https://ant.design/index-cn)

```shell
yarn add antd

nfo Direct dependencies
└─ antd@3.12.3
```

借助 [`babel-plugin-import`](https://www.npmjs.com/package/babel-plugin-import) 按需加载组件代码和样式：

```shell
# 安装
$ yarn add babel-plugin-import

info Direct dependencies
└─ babel-plugin-import@1.11.0
```

配置 `.babelrc.js`：

```javascript
module.exports = {
  presets: [
    "react-app"
  ],
  plugins: [
    // antD 按需加载
    [
      "import",
      {
        libraryName: "antd",
        // true - 加载源文件（less）；"css" - 加载 css 文件
        style: true
      }
    ]
  ]
};
```

使用：

```jsx
import { Button } from 'antd';

// ...
  render() {
    return (
      <div>
        <Button>点我</Button>
      </div>
    );
  }
// ...
```

## 3. 项目主页结构开发

### 3.1. 主页结构定义

* 页面结构定义：明白要做哪些页面，header/footer/menu 如何摆放。
* 目录结构定义：首先对业务的进行划分，才好在代码层面划分目录
* 栅格系统使用
* CSS3 `calc` 属性的使用

目录：

```text
react-ms/
  src/
    components/ # 公共组件
      Header/
        index.jsx
      Footer/
        index.jsx
      NavLeft/
        index.jsx
        index.module.less
    pages/      # 页面：引用组件
      Admin/      # 管理页面
        index.jsx
```

Admin/index.jsx：

```jsx
import React from 'react';
import { Row, Col } from 'antd';

import Header from '../../components/Header';
import Footer from '../../components/Footer';
import NavLeft from '../../components/NavLeft';

export default class Admin extends React.Component {
  constructor(props) {
    super(props);
    this.state = {};
  }

  render() {
    return (
      <Row>
        <Col span={3}>
          <NavLeft />
        </Col>
        <Col span={21}>
          <Header />
          <Row>
            内容区域
          </Row>
          <Footer />
        </Col>
      </Row>
    );
  }
}
```

公共样式 `src/style/common.less`：

