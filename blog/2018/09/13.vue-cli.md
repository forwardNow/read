# vue-cli 构建项目

## 安装

```shell
$ sudo npm install -g @vue/cli
+ @vue/cli@3.0.2
added 703 packages from 481 contributors in 34.835s

$ vue --version
3.0.2

$ sudo npm install -g @vue/cli-service-global
+ @vue/cli-service-global@3.0.2
```

$ vue create admin
